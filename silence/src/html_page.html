HTML

What is HTML?

HTML stands for Hypertext Markup Language and makes up the main structure of the code of a website

  • HTML syntax relies on tags
  • Tags are enclosed in angle brackets
  • End tags use a slash
How is it written?

All the code is written between the html and /html tags. Anything that appears on the main body of the website is placed between the body and /body tags

  • !DOCTYPE sets the document type as an HTML document
  • head defines the metadata of the page
  • body is the visible part of the website
