CSS

CSS can be directly linked to an HTML document. Using link rel=”stylesheet” href=”style.css”, a separate style.css file can be linked with the HTML, allowing for elements within the HTML code to have their styles modified within the CSS file. Things to note:
  • With this line, you are telling HTML you are linking a styling file
  • The link tag doesn't close
  • The file name of your CSS file should be within the quotes of href

In order to use CSS, you specify which element you want to change.

There are 3 ways you can do this:

  • You can change the style of all elements with a specific tag by using the tag name
  • You change the style of an id that was specified for an element by using a hashtag
  • You change the style of the class of an element using a period

There are many properties CSS can modify

These include:

  • color (American spelling)
  • background-color (also American spelling)
  • border
  • font-family
  • margin
  • padding
  • width
  • height
  • text-align
