What is JavaScript?

Javascript is a multi-paradigm and an Object-Oriented programming language

It also has the ability to manipulate HTML with the DOM API

Basic Syntax and Typing

Semicolons optional
JavaScript has 6 types of primitive variables:

  • null
  • undefined
  • numbers
  • strings
  • booleans
  • symbols

Variables

There are 3 ways to declare variables in JavaScript

  • let
    • "Standard" way to declare variables
    • Allows variables to be rebound later
    • Has block scope
  • var
    • Used between 1995-2015
    • Mostly now found in legacy code
    • Can store anything
    • Allows variables to be rebound later
    • Has function scope
    • const
      • Used to store constants
      • Unable to be rebound later
      • Has block scope

Functions

Functions are conceptually similar to methods in Java (though they have some different properties)
Functions can be declared using function() or as an "arrow function" using ()=>
